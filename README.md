# 🏏 AI Cricket Match Simulator (Cricbuzz-style)

An interactive **AI-powered cricket match simulator** built using modern frontend tools and intelligent logic.
Simulates ball-by-ball matches with dynamic commentary, realistic gameplay, and tournament modes.

---

## 🚀 Features

### 🧠 Smart Match Simulation

* Ball-by-ball cricket simulation
* Player vs Player logic (batsman vs bowler)
* Match phases (Powerplay, Middle Overs, Death Overs)
* Realistic scoring and wicket system

---

### 🤖 AI Commentary

* Dynamic commentary generation
* Context-aware match narration
* Fallback logic for consistent experience

---

### 🏆 Tournament Mode

* AI vs AI matches
* Round-robin fixtures
* Points table and standings
* Playoffs and final winner

---

### 🎨 Cricbuzz-style UI

* Clean and modern interface
* Dark theme with highlights
* Live scoreboard updates
* Commentary feed
* Last 6 balls tracking

---

## 🧱 Tech Stack

## ⚙️ How It Works

### Match Simulation Flow

1. User starts a match
2. Each ball is simulated using:

   * Player stats
   * Match context
3. Outcome is generated (runs or wicket)
4. Commentary is displayed
5. Score updates in real-time

---

### Simulation Logic

* Uses weighted probabilities
* Considers player strengths
* Adjusts for match situations
